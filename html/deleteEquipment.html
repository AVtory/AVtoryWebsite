Name: Delete Item
Primary Actors: Standard User or Administrator User Priority: High
Entry Condition: User has privileges to access inventory and is on the equipment page
Exit Condition: User successfully deletes an item
Flow of Events: User clicks on the “Delete Items” button on the screen.
User will select “View All Inventory” or “View Inventory by Category” of items.
User selects an item to be removed and deletes the item.
Alternative Flows: User inputs are invalid, prompt an error message with proper fields to enter.
User attempts to edit an item that does not exist.
Special Requirements: It takes no longer than 30 seconds to edit an existing item.
